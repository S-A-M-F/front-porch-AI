Front Porch AI

Private AI character chat — running entirely on your machine.

Free, open-source desktop app for immersive AI conversations. Run models locally for full privacy, or connect to OpenRouter and other remote APIs. Persistent memory, voice chat, and a relationship system that makes every character feel alive.

Everything you need, nothing you don't

🔒

Private by Default

Run AI models locally with KoboldCpp — your conversations never leave your machine. Or optionally connect to OpenRouter and other remote APIs. No accounts, no telemetry.

🧠

Realism Engine

Characters develop bonds, trust, and emotions across conversations. A dynamic relationship system that evolves with every interaction.

💬

Rich Character System

Full V2/V2.5 character card support. Import from SillyTavern, Chub.ai, or create from scratch with the built-in editor.

🎙️

Voice Chat

Full-duplex voice conversations with Whisper STT and Kokoro TTS. Push-to-talk or hands-free call mode.

📚

Long-Term Memory

RAG-based memory system remembers past conversations. Characters reference details from sessions days ago.

✍️

Story Generator

Multi-agent novel writing pipeline. Generate full stories with character arcs, world-building, and structured narratives.

☁️

Cloud Sync

Bi-directional sync to Google Drive or WebDAV. Your characters and chats available across all your devices.

🖥️

Cross-Platform

Native desktop app for Windows, macOS, and Linux. Apple Silicon optimized. GPU acceleration via CUDA, Metal, Vulkan, or ROCm.

🌐

Built-in Web Server

Access your chat from any browser on your network. Full REST API with PIN authentication for remote access.
